(constant) apiFetch

A fetch wrapper that automatically injects the Authorization header and handles JSON parsing and error throwing.
Source:
  • api.js, line 30
Examples
// GET request
const users = await apiFetch('/api/users')
// POST request
const post = await apiFetch('/api/posts', {
  method: 'POST',
  body: JSON.stringify({ title: 'Hello' }),
})
