# Learning how to draw graphics in Canvas 2D.

## Setup

The project is initialized with Vite and pnpm.

## Running

Typically running in a dev mode, as `pnpm run dev` or simply `pnpm dev` and
accessing entry points manually at their places/addresses (for example,
`http://localhost:5173/src/episodes/01-intro/index.html`).

## Layout

Code which is used by many entry points is placed in the `src/shared` folder.
